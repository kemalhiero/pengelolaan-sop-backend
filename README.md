# pengelolaan-sop-backend

Back-End Tugas Akhir "Sistem Informasi Pengelolaan SOP di Departemen Sistem Informasi Universitas Andalas berbasis web".

- NodeJS 22.11.0
- MariaDB 11.4.3

### Project Setup

```sh
npm install
```

### Project Development

```sh
npm run dev
```

### Run Project on Production

```sh
npm run start
```
